polire.interpolate.spline.bspline module¶

class polire.interpolate.spline.bspline.BSpline(kx=3, ky=3, s=None, resolution='standard', coordinate_type='Euclidean')¶

Bases: polire.interpolate.base.base.Base

Class to use a bivariate B-spline to interpolate values. https://docs.scipy.org/doc/scipy-0.18.1/reference/generated/scipy.interpolate.bisplrep.html#scipy.interpolate.bisplrep

kx, ky: int, int, optional

The degrees of the spline (1 <= kx, ky <= 5). Third order (kx=ky=3) is recommended.

sfloat, optional

A non-negative smoothing factor. If weights correspond to the inverse of the standard-deviation of the errors in z, then a good s-value should be found in the range (m-sqrt(2*m),m+sqrt(2*m)) where m=len(x).
